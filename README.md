# RishiHire - Premium Job Portal

**RishiHire** is a full-stack job portal built with Python and Django. It features a modern, responsive UI and a robust role-based system for Employers and Candidates.

##  Key Features
- **Dual User Roles:** Custom dashboards for both Candidates (Job Seekers) and Employers (Recruiters).
- **Job Management:** Employers can create, edit, and manage job postings.
- **Application Tracking:** Candidates can apply for jobs with PDF resumes.
- **Media Handling:** Automated resume storage and retrieval.
- **Premium UI:** Designed using Bootstrap 5 with a focus on User Experience (UX).

##  Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML5, CSS3, Bootstrap 5, Font Awesome
- **Database:** SQLite (Development)
